# flutter_eth_wallet

## Features



*支持通过生成助记词、Keystore文件、私钥 创建钱包账号。
 支持导出钱包账号助记词、私钥、Keystore文件。
 支持多个钱包账号管理
 账户余额查询及转账功能（二维码扫描支持）。
 支持ERC20 代币（余额显示、转账、代币币价显示）
 支持用法币（美元和人民币）实时显示币价。
 历史交易列表显示*
